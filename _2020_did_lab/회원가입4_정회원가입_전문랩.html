회원ID

  • * 아이디를 입력하신 후 중복확인을 클릭하세요.
  • * 아이디는 4~12자 이내로 영문 소문자와 숫자만 가능하며, 첫 자는 반드시 영문 소문자를 입력하여야 합니다.
  • * 한번 가입한 아이디는 변경이 불가하며, 탈퇴 후 재가입 시 동일 아이디 사용이 불가하므로 신중히 입력하여 주세요.
비밀번호

  • * 비밀번호는 영문, 숫자, 특수문자 중 2종류를 조합하여 최소 10자리 이상 또는 3가지 조합하여 8자리 이상으로 만들어 주세요.
  • * 특수문자 사용가능 범위: ! ” # $ % ( ) * + - / : ;<, -> ? @ [ ] ^ _ ‘ { | } ~ \ /
  • * 연속번호, 전화번호, 아이디 등 추측하기 쉬운 개인정보로 비밀번호를 만드실 수 없습니다.
